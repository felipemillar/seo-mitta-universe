## 📂 Estructura de Directorios (Propuesta y Actualizada)

### 01_Estrategia_y_Planes
Documentación fundacional y estratégica del proyecto.
- **Master_Plan_Entregables.md**: El documento maestro que define la metodología, KPIs y workflow del equipo.
- **Base_Conocimiento_Milanote.md**: Inteligencia estratégica, pilares y GEO guidelines.
- **Milanote_Raw_Export.md**: Exportación original del tablero para referencia histórica.
- **Plan_Integracion_Dashboard.md**: Estrategia para la visualización de datos y reporting.
- **Propuesta de Servicios SEO Integral para Mitta.pdf**: Documento comercial con el alcance legal y técnico.

Documentos técnicos generados para cada fase del proyecto (Redirecciones, Siloing, Schema, GEO, etc.).
- `Entregable_Fase1` a `Entregable_Fase8`.

### 03_Auditorias_y_Reportes
Auditorías técnicas (HTML/PDF) y reportes mensuales de rendimiento.
- **Mitta_SEO_Audit_Abril.html**: Auditoría técnica profunda de Abril.
- **Reporte_Auditoria_Mitta_Abril.pdf**: Resumen ejecutivo de auditoría.
- **Reportes_Abril_Mitta/**: Carpeta con data cruda y reportes adicionales.

### 04_Subproyectos_y_Pilotos
Iniciativas específicas y experimentación con IA.
- **Piloto_Hubspot/**: Integración CRM/SEO.
- **Pipeline_Renting_MittaGo/**: Workflow específico para la plataforma MittaGo.
- **geo-seo-claude/**: Experimentos de optimización para motores generativos (GEO).

### 05_Equipo_y_Skills
Recursos compartidos, carpetas personales de consultores y habilidades del agente.
- **Agentic-SEO-Skill/**: Habilidades programadas para el agente.
- **SEO_Gabriel/**, **SEO_Karen/**: Workspaces de consultoría.

## 🤖 Contexto para el Agente (Protocolo de Inicio)

Cada vez que se inicie este proyecto, el agente debe:
1.  **Sincronizar con NotebookLM**: Verificar acceso a los notebooks de "Market Analysis" e "INM Digital Predict".
2.  **Leer el Master Plan**: Asegurarse de entender los "Créditos SEO" y el modelo de gestión (1 crédito = 15 min).
3.  **Respetar la Línea Lógica**: InMedios **no** interviene el código; genera tickets detallados para TI/Desarrollo del cliente.
